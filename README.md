# M68k-Interp

A WIP m68k interpreter and disassembler

## Supported Instructions

The following instructions are currently supported by the decoder:

- `illegal`
- `rte`
- `rts`
- `rtr`
- `nop`
- `trapV`
- `reset`

- `jsr`
- `jmp`

- `add`
- `adda`
- `addx`

- `tas`
- `tst`

- `asd`
- `lsd`
- `roxd`
- `rod`

- `negx`
- `clr`
- `neg`
- `not`

- `trap`
- `lnk`
- `unlk`

- `btst`
- `bchg`
- `bclr`
- `bset`

- `sub`
- `suba`
- `subx`

- `addi`
- `andi`
- `subi`

- `eori`
- `cmpi`

- `ori`

- `movep`
- `movea`
- `move`

- `ext`
- `nbcd`
- `swap`
- `pea`

- `movem`
- `lea`
- `chk`

- `addq`
- `subq`
- `moveq`
