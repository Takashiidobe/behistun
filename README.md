# M68k-Interp

A WIP m68k interpreter and disassembler

## Supported Instructions

The following instructions are currently supported by the decoder:

- `MOVE`
- `MOVEA`
- `MOVEQ`
- `NEGX`
- `NEG`
- `NOT`
- `MOVE from SR`
- `SUBA`
- `ADDA`
- `CMPA`
- `PEA`
- `CLR`
- `LEA`
- `LINK`
- `UNLK`
- `BSR`
- `BRA`
- `BHI`
- `BLS`
- `BCC`
- `BCS`
- `BNE`
- `BEQ`
- `BVC`
- `BVS`
- `BPL`
- `BMI`
- `BGE`
- `BLT`
- `BGT`
- `BLE`
- `BTST`
- `BCHG`
- `BCLR`
- `BSET`
- `MOVEM`
- `ADDQ`
- `SUBQ`
- `LSL`
- `LSR`
- `ORI`
- `OR`
- `ANDI`
- `SUBI`
- `ADDI`
- `EORI`
- `CMPI`
- `CMP`
- `TST`
- `RTS`
